 Ultrasonic Radar System

 Overview

The Ultrasonic Radar System is a simple project that uses ultrasonic waves to detect objects and measure distance. It works like SONAR by sending sound waves and calculating the time taken for the echo to return.



 Objective

* Detect objects in real-time
* Measure distance using ultrasonic waves
* Display results in a radar-style interface
* Build a low-cost, working prototype



 Components Used

* Arduino (ATmega328)
* Ultrasonic Sensor (HC-SR04)
* Servo Motor
* Breadboard & Jumper Wires
* Power Supply



 How It Works

1. The ultrasonic sensor sends sound waves
2. Waves hit an object and reflect back
3. Arduino calculates distance using echo time
4. Servo motor rotates the sensor (0°–180°)
5. Data is displayed as a radar visualization



 Features

* Real-time object detection
* Distance measurement
* 180° scanning
* Radar-style display



📊 Applications

* Obstacle avoidance (robots, cars)
* Parking assistance
* Security systems
* Smart home automation



⚠️ Limitations

* Short range (2–40 cm)
* Not a real RF radar (uses sound waves)
* Limited to 180° scanning



🛠️ Tech Stack

* Arduino IDE (C/C++)
* Processing (Java) for GUI





✅ Outcome

A working ultrasonic radar prototype that detects objects, measures distance, and visualizes data in real time.
